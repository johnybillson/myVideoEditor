========================================================================
   DLL DE CONTRÔLE ACTIVEX : Vue d'ensemble du projet 
   myVideoEditor
========================================================================

ControlWizard a créé ce projet pour votre DLL de contrôle ActiveX 
myVideoEditor, qui contient 1 contrôle.

Ce squelette de projet décrit non seulement les bases de l'écriture d'un
contrôle ActiveX, mais est également un point de départ pour l'écriture de 
fonctionnalités spécifiques à votre contrôle.

Ce fichier contient un résumé des éléments contenus dans chaque fichier qui
constitue votre DLL de contrôle ActiveX myVideoEditor.

myVideoEditor.vcxproj
    Il s'agit du fichier projet principal pour les projets VC++ générés à 
    l'aide d'un Assistant Application. Il contient des informations sur la 
    version de Visual C++ qui a généré le fichier, ainsi que des informations 
    sur les plateformes, configurations et fonctionnalités du projet que vous 
    avez sélectionnées à l'aide de l'Assistant Application.

myVideoEditor.vcxproj.filters
    Il s'agit du fichier de filtres pour les projets VC++ générés à l'aide d'un 
    Assistant Application. 
    Il contient des informations sur l'association entre les fichiers de votre 
    projet et les filtres. Cette association est utilisée dans l'IDE pour 
    afficher le regroupement des fichiers qui ont des extensions similares sous 
    un nœud spécifique (par exemple, les fichiers ".cpp" sont associés au 
    filtre "Fichiers sources").

myVideoEditor.h
    Il s'agit du fichier Include principal pour la DLL du contrôle ActiveX. Il
    comprend d'autres fichiers Include spécifiques au projet tels que 
    resource.h.

myVideoEditor.cpp
    Il s'agit du fichier source principal qui contient le code pour 
    l'initialisation de la DLL, sa finalisation et d'autres opérations.

myVideoEditor.rc
    Il s'agit de la liste des ressources Microsoft Windows que le projet
    utilise. Ce fichier peut être modifié directement dans l'éditeur de 
    ressources Visual C++.

myVideoEditor.def
    Ce fichier contient des informations relatives à la DLL du contrôle 
    ActiveX qui doit être exécutée avec Microsoft Windows.

myVideoEditor.idl
    Ce fichier contient le code source ODL (Object Description Language) 
    pour la bibliothèque de types de votre contrôle.

myVideoEditor.ico
    Ce fichier contient une icône qui s'affichera dans la boîte de dialogue 
    À propos de. Cette icône est incluse dans le fichier de ressources 
    principal myVideoEditor.rc.

Autres fichiers standard :

stdafx.h, stdafx.cpp
    Ces fichiers sont utilisés pour générer un fichier d'en-tête précompilé 
    (PCH) nommé myVideoEditor.pch et un fichier de types précompilés 
    (PCT) nommé stdafx.obj.

resource.h
    Il s'agit du ficher d'en-tête standard, qui définit les nouveaux ID de 
    ressources.
    L'éditeur de ressources Visual C++ lit et met à jour ce fichier.
